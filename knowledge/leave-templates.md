# 请假话术模板

## 公司场景

### 正式（邮件）
```
{领导称呼}您好，

我因{病症}身体不适，{今天/明天}需要请假{天数}天休息。

目前工作安排如下：
- {紧急事项}已交接给{同事名}
- {非紧急事项}我恢复后会跟进

如需紧急联系，可通过微信或电话联系我。

给您和团队带来不便，非常抱歉。

{姓名}
{日期}
```

### 半正式（微信/钉钉）
```
领导好，我今天{病症}不太舒服，想请{天数}天假休息一下。
工作已经交接好了，有事随时微信找我。麻烦批准，谢谢！
```

### 随意（关系好的领导）
```
老板，今天身体不太行，请假{天数}天🤒
有事微信滴我~
```

## 学校场景

### 正式
```
老师您好，

我是{班级}的{姓名}，因{病症}需要请假{天数}天。
落下的课程我会在恢复后补上。

请予以批准，谢谢。

{姓名}
{日期}
```

### 辅导员/班主任（微信）
```
老师好，我今天{病症}不舒服，想请假{天数}天去休息/看医生。
作业我会补上的，麻烦老师批准～
```

## 病症描述映射

不同病症对应的常用说法：
- 感冒+发烧 → "感冒发烧，身体不适"
- 肠胃炎 → "急性肠胃炎，上吐下泻"
- 偏头痛 → "偏头痛发作，需要休息"
- 咽炎 → "嗓子发炎，说话困难"
- 腰痛 → "腰部扭伤/劳损，行动不便"
- 过敏 → "严重过敏反应，需要就医"

## 日期相关建议

- 请1天：强调"休息一天就能恢复"
- 请2天：强调"需要观察/复查"
- 请3天以上：建议提"遵医嘱休息X天"
